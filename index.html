1. Question 1

    Use the simulator to read the data stored in address 10 of memory. In what direction is data sent on the data bus during this process?

    Hint: Memory uses the data bus to transport the data that was read to the processor.

  2. Question 2

    Use the simulator to read the data stored in address 8 of memory. In what direction is data sent on the address bus during this process?

    Hint: Address information always comes from the processor as it is the part responsible for following instructions.

  3. Question 3

    Use the simulator to write “11110000” into address 10 of memory. In what direction is data sent on the data bus during this process?

    Hint: The processor uses the data bus to transport the data meant for storage to memory.

  4. Question 4

    Use the simulator to write “00001111” into address 8 of memory. In what direction is data sent on the address bus during this process?

    Hint: Address information always comes from the processor as it is the part responsible for following instructions.

  5. Question 5

    The data bus is bi-directional because data can be sent in both directions between the processor and memory on the data bus. True or false?

    Hint: The same data bus is used for both writing data to memory and reading data from memory.

  6. Question 6

    The address bus is uni-directional because address information is always sent in one direction only from the processor to memory. True or false?

    Hint: Data on the address bus is never sent from memory to the processor.
